# CRUD de Alunos - Teste Técnico

Este projeto consiste na implementação de um CRUD de Alunos, utilizando **Angular** para o frontend e uma API em **C#** para o backend. O banco de dados pode ser escolhido livremente.

## 📌 Requisitos

- Desenvolver um CRUD de alunos.
- O **frontend** deve ser feito em **Angular**.
- O **backend** deve ser feito SE POSSIVEL em **C#**.
- O banco de dados pode ser escolhido livremente.
- **Autenticação não é necessária**.
- O código deve ser disponibilizado em um **repositório público no GitHub**.
